Para analisar a associação entre duas variáveis, pode-se representar os valores observados na amostra em gráficos bidimensionais adequados a cada tipo de variável.

A figura ao lado mostra um gráfico de box plot com dados da altura da população brasileira com idade de 14 anos. Com esse tipo de gráfico, pode-se analisar uma amostra de dados de uma variável quantitativa, representada, geralmente, no eixo vertical.

Cada segmento horizontal que forma o box plot representa uma medida de tendência central ou um quantil do conjunto de dados da variável, a saber: mínimo, quantil 25%, mediana, quantil 75% e máximo.

Você pode verificar a medida representada por cada segmento por meio do botão "Mostrar legenda".

Essa figura ficará disponível na referência 2 durante todo o software, para que você possa consultá-la quando achar necessário.
