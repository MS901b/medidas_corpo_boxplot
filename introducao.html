Mapa Introdução Início

A Estatística Descritiva tem como objetivo principal a construção de ferramentas que permitam extrair de um conjunto de dados toda informação relevante para a análise do fenômeno em estudo. Muitas dessas ferramentas permitem resumir os dados observados. São os casos dos gráficos, das tabelas e das medidas resumos numéricas.

Utilizando algumas dessas ferramentas, este software explora a comparação de conjuntos de dados referentes à altura e ao número do calçado de meninos e meninas.

Na Atividade 1, os dados que você tem em mãos pelos alunos devem ser registrados em uma tabela.

Na Atividade 2, esses dados são representados graficamente no chamado “gráfico de caixa” (box plot) para cada uma das categorias (meninas e meninos). Isso permite uma comparação visual da variável analisada (altura). Também serão comparadas algumas medidas resumos utilizadas na construção dos gráficos, como mediana, mínimo, máximo, e quartis.

Na Atividade 3, será construído outro box plot, para comparação visual da variável “número de calçado” de cada uma das categorias (meninas e meninos).
